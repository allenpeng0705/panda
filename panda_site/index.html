Panda AI Gateway

A high-performance, stateless, intent-aware gateway for LLMs and AI services

Panda is a Rust-based AI gateway that sits between your clients and model providers, adding policy, identity, budgets, MCP tools, caching, safety, and observability in one place.

Key Capabilities

Security & Safety

JWT/JWKS authentication, prompt safety, PII scrubbing, and Wasm plugins for custom security policies.

Cost Control

TPM-style token budgets, semantic caching, and detailed usage metrics to optimize AI spending.

MCP Tools

Model Context Protocol for tool orchestration, enabling agents to use external services seamlessly.

Observability

Prometheus metrics, OpenTelemetry tracing, and structured logs for comprehensive monitoring.

Extensibility

Wasm plugins for custom policy, transforms, and integrations without forking the core.

Performance

Rust-based implementation with async I/O, minimal copying, and optimized streaming paths.

Four-Layer Design

  • Ingress Layer - Accepts connections and normalizes protocols
  • Shield Layer - Security and hygiene before downstream work
  • Intelligence Layer - Intent, routing, reuse, and tool surfacing
  • Egress Layer - Adapts to each backend's format
